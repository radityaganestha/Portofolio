Web Calculator

Web Pencari Film

Web ini adalah aplikasi pemfilteran film yang menampilkan film populer dari The Movie Database (TMDb). Pengguna dapat mencari film berdasarkan judul, dengan setiap film menampilkan poster, judul, rating (hijau, oranye, merah), dan ringkasan singkat. Desain responsif dan antarmuka sederhana memberikan pengalaman yang menyenangkan, dengan hasil pencarian diperbarui otomatis. Aplikasi ini menggunakan HTML, CSS, dan JavaScript.

Biodata Kelas XI RPL 4

Web ini menampilkan tabel biodata siswa kelas XI RPL 4, mencakup nama, agama, jenis kelamin, alamat, nilai tugas, perilaku, dan hobi. Nilai diberikan dengan kode warna: Merah (K) untuk kurang, Kuning (B) untuk baik, dan Hijau (SB) untuk sangat baik. Terdapat juga kolom untuk tanda tangan pengajar sebagai validasi. Desain tabel yang rapi memberikan gambaran menyeluruh tentang siswa dalam tampilan yang mudah dibaca.

Form Pendaftaran Siswa Online

Web ini adalah formulir pendaftaran siswa online untuk mengumpulkan informasi penting seperti nama, jenis kelamin, email, nomor telepon, kota domisili, asal sekolah, dan jurusan pilihan. Dengan desain menarik dan animasi pada judul, formulir ini memudahkan calon siswa untuk mendaftar secara efisien, dengan kolom wajib diindikasikan oleh tanda bintang (*).

Pengecekan Bilangan Prima C++

Program C++ ini memeriksa apakah sebuah angka yang dimasukkan oleh pengguna adalah angka prima. Jika bukan, program menampilkan faktor-faktornya. Setelah itu, pengguna dapat memilih untuk mengulang program dengan mengetik 'Y' atau 'y', atau mengakhiri program dengan 'N' atau 'n'. Program juga membersihkan layar di setiap iterasi dan menangani input yang tidak valid menggunakan pernyataan goto.

Program Kasir C++

Program C++ ini adalah sistem kasir sederhana yang memungkinkan pengguna memasukkan jumlah barang, nama, jumlah, dan harga masing-masing. Setelah menghitung total harga, pengguna diminta memasukkan uang yang dibayarkan, dan program menampilkan kembalian atau informasi jika uang tidak mencukupi, diakhiri dengan ucapan terima kasih.

Program Rapot C++

Program C++ ini mengelola rapor nilai siswa dengan fitur registrasi dan login. Pengguna memasukkan nama, kelas, absen, username, dan password. Setelah login berhasil, siswa dapat memasukkan nilai untuk berbagai mata pelajaran, yang dihitung total, tertinggi, dan terendahnya, beserta status kelulusan. Program memungkinkan pengguna untuk memasukkan nilai lagi atau melihat ringkasan hasil, termasuk rata-rata nilai.
